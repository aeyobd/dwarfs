### DELVE
We retrieve DELVE through topcat's TAP query service using the query
```
SELECT * FROM delve_dr2.objects
WHERE ra < 209.3 + 4
and ra > 209.3 - 4
and dec < 26.8 + 3
and dec > 26.8 - 3
```

```
SELECT * FROM delve_dr2.objects 
where mag_psf_g < 99
and magerr_psf_g < 99
and mag_psf_r < 99
and magerr_psf_r < 99
and q3c_radial_query(ra, dec, 209.3, 26.8, 6)
```
